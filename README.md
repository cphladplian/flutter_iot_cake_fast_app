## 🍰 fastCake – Cake Shop App
แอปพลิเคชันแสดงร้านเค้กในกรุงเทพฯ พร้อมรายละเอียดร้าน รูปภาพ ช่องทางติดต่อ และแผนที่ 📍

## 📱 Features
- 🎞️ แสดงรูปเค้กแบบ Carousel (เลื่อนอัตโนมัติ)
- 📋 แสดงรายการร้านเค้ก
- 📄 ดูรายละเอียดร้านแบบครบถ้วน
- 📞 โทรหาร้านได้ทันที
- 🌐 เปิดเว็บไซต์ / Facebook
- 🗺️ แสดงแผนที่ร้าน + เปิด Google Maps

## 🔄 App Flow (Flow 1–5)
### Flow 1: Splash Screen
- แสดงหน้าโหลด 3 วินาที
- ไปหน้า List อัตโนมัติ

### Flow 2: Cake Shop List
- แสดง Carousel รูปเค้ก
- แสดงรายการร้านทั้งหมด

### Flow 3: Select Shop
- ผู้ใช้เลือกร้าน
- ส่งข้อมูลไปหน้า Detail

### Flow 4: Shop Detail
แสดง:
- รูปภาพร้าน
- ชื่อร้าน
- ที่อยู่
- เบอร์โทร
- Website / Facebook

### Flow 5: Map & Navigation
- แสดงตำแหน่งร้านบนแผนที่
- กด Marker เพื่อเปิด Google Maps
